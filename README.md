# Название проекта:  «Рыболовная флотилия»

# Описание

Проект для отслеживания улова по траулерам и членам экипажа. Обеспечиваются операции insert, update, delete для базы данных. При добавлении информации про результаты рейса происходит автоматическое обновление статистики

# Установка

**Ubuntu**
```bash
git clone git@github.com:DendroidBrunch/tpmp-lab4-rep2.git
cmake -B build
cmake --build build
```

# Usage

```
bash
./build/fishing_fleet
```
